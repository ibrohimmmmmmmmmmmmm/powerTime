# ⚡ PowerTime

PowerTime is a modern **RF Battery Runtime Analyzer Tool** built to estimate battery performance based on real-world usage patterns like TX/RX/Idle duty cycles. It provides accurate runtime predictions with a clean and responsive UI.

---

## 🚀 Features

- 📊 **Battery Runtime Estimation**
- 📡 **RF Duty Cycle Analysis (TX / RX / Idle)**
- 🔋 **Average Current Calculation**
- ⏱️ Real-time **runtime conversion (Hours / Minutes)**
- 🌙 Beautiful **Dark / Light UI support**
- ⚡ Fast and responsive interface
- 🌍 Full **i18n support (multi-language ready)**
- 🎯 Professional UX designed for RF engineers and technicians

---

## 🛠️ Tech Stack

PowerTime is built using modern frontend technologies:

- ⚛️ **React + TypeScript (TS / TSX)**
- 🎨 **Tailwind CSS** for styling
- 🧩 **Lucide React Icons**
- ⚡ **Vite / Fast build system**
- 🌐 **i18n internationalization system**
- 🧠 Clean component-based architecture
